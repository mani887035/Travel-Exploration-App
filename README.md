# 🌍 Travel Exploration App

A modern and interactive travel exploration web application built using **Antigravity AI**. This app helps users discover beautiful destinations, explore travel experiences, and plan their next adventure with an engaging and user-friendly interface.

## 🚀 Live Demo
🔗 https://mani887035.github.io/Travel-Exploration-App/

## 📌 Features

- 🌎 Explore popular travel destinations  
- 🏝️ Beautiful and responsive UI design  
- 📱 Mobile-friendly experience  
- 🔍 Easy navigation for discovering places  
- ✨ Smooth animations and interactive sections  
- 🧭 Travel inspiration for users

## 🛠️ Built With

- HTML
